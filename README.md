# Servicio Web de Contactos

API desarrollada con Node.js y Express que permite:

## Endpoints

- GET /contactos → Lista contactos
- POST /contactos → Guarda contactos

## Tecnologías
- Node.js
- Express
- Axios

## Ejecución
npm install
node index.js
